This directory contains the Nios2 software.

To import project into Eclipse:
1) Start eclipse either through Quartus II, or by finding and running the
   executable in the altera installation folder.
2) Go to File->Import->General->Existing Projects into Workspace.
3) Click Next and select the dtb_expert folder in the software.
4) Click Finish.
5) Repeat the same procedure for dtb_bsp.
